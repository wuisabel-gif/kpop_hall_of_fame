# Kpop Wiki

🇬🇧 [Read in English](README.md)

![데모](demo.gif)

위키백과 표처럼 훑어보는 게 아니라, 진짜로 걸어 다니는 느낌으로 K-pop 역사를 둘러보고 싶었다. 그래서 만들었다.

어두운 복도에서 시작한다. 그룹마다 빛나는 문이 하나씩 있고 — 입구엔 신화, 맨 끝엔 KATSEYE — 위로 스크롤하면 그 문들을 지나 걷게 된다. 한 세대에서 다음 세대로 넘어갈 때마다 조명 색이 바뀐다. 걷는 동안 K-Pop ON! 플레이리스트가 흘러나온다. 문을 누르면 카드가 뒤집히면서 데뷔일, 소속사, 멤버 수, 팬덤 이름이 보인다. 사진은 진짜다 — 스포티파이에서 실시간으로 불러온다.

복도 끝에 닿으면 카탈로그로 들어간다 — 5세대 전체가 펼쳐지고, 누가 언제 데뷔했는지, 날짜와 티켓 링크가 달린 2026 투어 보드, 그리고 방금 나온 신곡까지.

## 실행하기

`index.html` 파일을 열기만 하면 된다. 그게 전부다 — 파일 하나, 빌드 없음, 설치 없음. 사진과 음악은 인터넷 연결이 필요하다.

## 솔직한 메모 몇 가지

- 투어 날짜, 티켓 상태, "신곡"은 2026년 6월 기준으로 내가 직접 채워 넣은 스냅샷이다. 진리가 아니라 분위기로만 봐 달라 — 표 사기 전에 실제 판매처를 꼭 확인하길.
- 전체 곡은 브라우저에서 스포티파이에 로그인돼 있을 때만 재생된다. 아니면 30초 미리듣기만 나온다. 내 규칙이 아니라 스포티파이 규칙이다.
- 한 세대가 어디서 끝나고 다음이 어디서 시작되는지는 과학이 아니라 팬들의 논쟁이다. 뉴진스 하나만으로도 싸움 난다. 다들 긋는 자리에 선을 긋고 계속 걸었다.

재미로 만들었다. 포크하든, 부수든, 네 취향을 더하든.
